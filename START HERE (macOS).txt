Oasis for macOS - start here
==================================

EASIEST (recommended) - install from Terminal, no Gatekeeper prompts:

   1. Open Terminal (Applications > Utilities > Terminal, or Spotlight "Terminal").
   2. Paste this line and press Return:

        curl -fsSL https://dakota1450.github.io/OASIS/install.sh | bash

   It downloads Oasis, makes a Desktop launcher, and opens it. Because the files
   come from Terminal (not a browser download), macOS does NOT quarantine them,
   so there is no "unidentified developer" block to fight.


IF YOU DOWNLOADED THIS ZIP INSTEAD

A zip downloaded in a browser is quarantined by macOS, and recent macOS
(Sequoia+) blocks the launchers with NO "Open Anyway" button. Clear the
quarantine and make the launchers runnable - in Terminal, run once:

        xattr -dr com.apple.quarantine "$(pwd)" ; chmod +x *.command

   (drag this folder onto the Terminal window first so it's the current folder),
   then double-click  "Setup Oasis.command".

   Or just run the one-line installer above - it replaces all of this.


Requirements
------------
   * macOS (Apple silicon or Intel)
   * Node.js - free, from https://nodejs.org/en/download  (the installer checks)
   * Claude Code + Codex recommended, not required

Everything Oasis stores stays on this machine. No account, no telemetry.
Open Oasis any time from the Desktop launcher, or run "Launch Oasis.command".
After installing an update, run "Restart Oasis.command" once so the new
version actually loads (Launch only reuses an already-running copy).
